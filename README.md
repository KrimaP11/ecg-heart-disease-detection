# ECG Heart Disease Detection Using Deep Learning

## Project Overview

This project focuses on early detection of myocardial infarction (heart attack) using artificial intelligence and deep learning techniques applied to ECG (Electrocardiogram) signals.

The project is based on my published research paper:

**AI-Based ECG Analysis for Early Myocardial Infarction Detection**

Published in:
International Journal of Scientific Research in Computer Science, Engineering and Information Technology (IJSRCSEIT)

DOI:
https://doi.org/10.32628/CSEIT2410790

---

## Research Objective

The objective of this project is to develop a deep learning system capable of detecting myocardial infarction from ECG recordings with high accuracy and reliability.

## Performance

### ResNet1D

- Accuracy: 96.30%
- AUROC: 0.981

### CNN-BiLSTM-Attention

- Accuracy: 95.20%
- AUROC: 0.981
